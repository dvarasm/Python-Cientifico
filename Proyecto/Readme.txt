Para poder ejecutar la aplicación se deben descargar los modelos Yolo y retina net
de la siguiente página https://github.com/OlafenwaMoses/ImageAI/releases/tag/1.0/
ahí descargar los archivos Yolo.h5 y resnet50_coco_best_v2.0.1.h5
y ponerlos en la misma carpeta que esta el código.
Luego ejecutar python MotorDetector.py

Ambos modelos no se adjuntan ya que yolo pesa 237mb y retinanet pesa 146mb
